# Simple Calculator

A continuous calculator program that performs basic arithmetic operations with an infinite loop.

## Description

This is a command-line calculator that runs in a continuous loop, allowing users to perform multiple calculations without restarting the program. It supports addition, subtraction, multiplication, division, and exponentiation.

## Features

- Continuous calculation loop (runs until manually stopped)
- Supports 5 operations: +, -, *, /, ** (power)
- Handles floating-point numbers
- Error handling for invalid operators
- Clear visual separation between calculations

## Requirements

- Python 3.x
- No external dependencies (uses only standard library)
