# Age Calculator

A simple Python script that calculates and displays your age in years, months, and days.

## Description

This program takes a user's name and age in years, then calculates the equivalent age in months and total days lived, accounting for leap years.

## Features

- Accepts user name and age input
- Calculates age in months and days
- Automatically accounts for leap years
- Uses system's current date for accurate calculations

## Requirements

- Python 3.x
- No external dependencies (uses only standard library)
