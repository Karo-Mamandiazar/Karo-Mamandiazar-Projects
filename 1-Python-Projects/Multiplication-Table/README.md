# Multiplication Table Generator

A simple Python program that generates a complete multiplication table from 1 to 10.

## How It Works

The program uses nested loops to calculate and display all multiplication combinations:
- Outer loop: Runs from 1 to 10 (the first number)
- Inner loop: Runs from 1 to 10 (the second number)
- Displays each multiplication equation
- Adds a separator line after each row
