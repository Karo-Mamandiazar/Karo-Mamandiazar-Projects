# Random Password Generator

A Python program that generates random passwords using NumPy by randomly selecting characters from a predefined set.

## How It Works

The program:
1. Creates an array of allowed characters (letters, numbers, and special characters)
2. Randomly selects 12 characters from the array
3. Joins them together to form a password
4. Displays the character set and the generated password

## Requirements
```
numpy>=1.15.0
```
