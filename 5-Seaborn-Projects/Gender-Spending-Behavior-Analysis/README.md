# Gender Spending Behavior Analysis

A Python program that analyzes and visualizes spending behavior between different genders using Seaborn and Matplotlib.

## Features

- Loads the restaurant tips dataset from Seaborn
- Compares total bill amounts by gender
- Creates a boxplot visualization
- Identifies differences in customer payment patterns

## How It Works

The program:
1. Loads the built-in tips dataset from Seaborn
2. Creates a boxplot comparing total bill by gender
3. Visualizes the distribution of spending for each gender
4. Displays the comparison chart

## Dataset Information

The **tips dataset** contains restaurant customer information:
- **total_bill**: Total bill amount
- **tip**: Tip amount
- **sex**: Gender of customer (Male/Female)
- **smoker**: Whether customer smokes
- **day**: Day of the week
- **time**: Lunch or Dinner
- **size**: Party size
