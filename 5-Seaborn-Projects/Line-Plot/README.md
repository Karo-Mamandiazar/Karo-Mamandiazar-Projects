# Line Plot with Seaborn

A Python program that creates a line plot to visualize trends over time or continuous data using Seaborn.

## Features

- Loads the flights dataset from Seaborn
- Creates a line plot of passenger count by year
- Shows trends over time
- Groups data by month

## How It Works

The program:
1. Loads the flights dataset
2. Aggregates passenger counts by year
3. Creates a line plot showing passenger trends

## Line Plot Explained

A line plot shows data points connected by lines:
- Best for time series or continuous data
- Shows trends and patterns
- Can display multiple groups

## Requirements
```
seaborn>=0.11.0
matplotlib>=3.0.0
```
