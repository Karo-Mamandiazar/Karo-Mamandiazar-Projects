# Bar Plot with Seaborn

A Python program that creates a bar plot to visualize categorical data using Seaborn.

## Features

- Loads the tips dataset from Seaborn
- Creates a bar plot of average tip by day
- Shows aggregated values
- Compares categories

## How It Works

The program:
1. Loads the tips dataset
2. Calculates average tip for each day
3. Creates a bar plot showing the comparison

## Bar Plot Explained

A bar plot shows categorical data:
- Each bar represents a category
- Bar height shows values
- Great for comparisons

## Requiremets
```
seaborn>=0.11.0
matplotlib>=3.0.0
```
