# Histogram with Seaborn

A Python program that creates a histogram to visualize the distribution of a single numerical variable using Seaborn.

## Features

- Loads the tips dataset from Seaborn
- Creates a histogram of total bill amounts
- Shows frequency distribution
- Adds KDE (Kernel Density Estimate)

## How It Works

The program:
1. Loads the tips dataset
2. Creates a histogram of total_bill values
3. Shows the distribution of bill amounts

## Histogram Explained

A histogram shows frequency distribution:
- Groups data into bins
- Height shows frequency
- Reveals distribution shape

## Requirements
```
seaborn>=0.11.0
matplotlib>=3.0.0
```
